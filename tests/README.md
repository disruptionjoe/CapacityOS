# Tests

This folder defines the frequent-task acceptance process for CapacityOS.

The goal is not unit coverage first.

The goal is confidence that Joe's common behaviors work cleanly before any real
migration happens.

## Test waves

- **Wave 1** - dummy seeded content
- **Wave 2** - one real imported sample from the current system

## Pass conditions

- Cowork and Codex land on the same lifecycle outcome
- engine repo remains clean while runtime mutates
- no manual queue bridge is required
- Joe-facing outputs stay simple
