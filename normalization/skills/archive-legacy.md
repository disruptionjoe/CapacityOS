# Skill: archive-legacy

## Purpose

Move legacy material into reference/archive homes without promoting it into
active state.

## Input

- classified archive candidate

## Output

- archived legacy record

## Steps

1. Preserve original path in provenance.
2. Store content in the correct archive/reference area.
3. Tag it with project and era if known.
4. Leave searchable breadcrumbs.

## Rules

- Archiving is not deletion.
- Historical naming can remain historical.
